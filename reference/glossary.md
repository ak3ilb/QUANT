# Glossary

| Term | Definition |
|------|------------|
| **ATR** | Average True Range — volatility measure |
| **ANM-MM** | Additive Noise Model Mixture Model — causal inference framework |
| **BOCPD** | Bayesian Online Changepoint Detection — detects regime breaks in streaming data |
| **Basis** | Futures price minus spot price |
| **CID** | Complexity-Invariant Distance — time series distance correcting for pattern complexity |
| **Discord** | Most anomalous subsequence in a time series (Matrix Profile max) |
| **DTW** | Dynamic Time Warping — aligns time series with non-linear warping |
| **DRL** | Deep Reinforcement Learning |
| **FII** | Foreign Institutional Investor (India) |
| **DII** | Domestic Institutional Investor (India) |
| **FinRL** | Open-source financial RL framework |
| **Funding rate** | Periodic payment on crypto perps (longs pay shorts or vice versa) |
| **Gamma** | Rate of change of delta (options); extreme on expiry |
| **HMM** | Hidden Markov Model — latent regime detection |
| **IST** | Indian Standard Time (UTC+5:30) |
| **Lot size** | Contract multiplier (Nifty 65, Bank Nifty 30) |
| **Matrix Profile** | Vector of nearest-neighbor distances for all subsequences |
| **Motif** | Recurring similar subsequence pair in time series |
| **MPC** | Monetary Policy Committee (RBI) |
| **P&L** | Profit and Loss |
| **Perp** | Perpetual futures (no expiry) |
| **Pip** | Minimum price increment (XAUUSD: $0.01) |
| **SAX** | Symbolic Aggregate approXimation — discretize time series to symbols |
| **SEBI** | Securities and Exchange Board of India |
| **Shapelet** | Discriminative subsequence for classification |
| **Sharpe ratio** | Risk-adjusted return: (R − Rf) / σ |
| **SLIM** | Side-Length-Independent Motif discovery |
| **STT** | Securities Transaction Tax (India) |
| **STOMP** | Scalable Time series Ordered-search Matrix Profile |
| **STUMPY** | Python library for Matrix Profile computation |
| **TS-K-means** | Time Series K-means with DTW distance |
| **VALMOD** | Variable Length Motif Discovery |
| **VIX / IndiaVIX** | Implied volatility index (US / India) |
| **Walk-forward** | Rolling train/test to avoid overfitting |
| **Warping path** | Alignment path through DTW cost matrix |
| **Z-normalize** | (x − μ) / σ per subsequence before distance comparison |
